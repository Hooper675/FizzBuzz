This is the challenge for the week. I made a couple of mistakes while testing out my code. For instance, the condition that number i is divisible by both 3 and 5 was placed at the bottom, after the condition to check whether number i is divisible by 5. 
Essentially, I wrote it like this: if (i % 3 == 0)...i %5 and then that statement. 
There was no output because all three of these conditional statements overlap with each other. So, it's best to place the more specific conditional statement first, followed by the least specific ones. 
In coding, the order of operations is a bit different.
